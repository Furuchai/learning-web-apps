## 📂 プロジェクト一覧

### 1. To-Do App
ローカルストレージを使用した、タスク管理アプリケーションです。

![ToDoアプリのスクリーンショット](images/todo-app-screenshot.png)

* **使用技術:** HTML5, CSS3, JavaScript (Vanilla), LocalStorage
* **機能:**
    * タスクの追加・削除
    * 完了状態の切り替え（取り消し線）
    * データの永続化（ブラウザを閉じても保存）
    * レスポンシブデザイン（スマホ対応）
