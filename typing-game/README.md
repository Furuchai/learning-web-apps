# Simple Typing Game ⌨️

JavaScript学習用のシンプルなタイピングゲームです。
制限時間内に表示される英単語をどれだけ正確に入力できるかを競います。

## 🎯 機能 (MVP)
* **ランダム出題:** 用意された単語リストからランダムに表示。
* **入力判定:** 入力された文字列とお題の一致を判定。
* **スコア機能:** 正解数をカウント。
* **タイマー:** 制限時間（60秒）のカウントダウンとゲーム終了判定。

## 🛠 使用技術
* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 使い方
1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開きます。
3. 表示される単語を入力フォームに入力してください。
